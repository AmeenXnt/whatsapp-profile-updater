# WhatsApp Profile Updater

A web application that allows users to update their WhatsApp profile picture securely using QR code or pairing code authentication.

## Features

- QR Code Authentication
- Pairing Code Authentication
- Secure Profile Picture Update
- Modern UI with Glass Design
- Auto-logout after update
- Mobile Responsive

## Tech Stack

- Node.js
- HTML/CSS
- TailwindCSS
- WhatsApp Web API

## Installation

1. Clone the repository
```bash
git clone <your-repo-url>
cd whatsapp-profile-updater
```

2. Install dependencies
```bash
npm install
```

3. Start the server
```bash
npm start
```

## Usage

1. Open the application in your browser
2. Choose authentication method (QR Code or Pairing Code)
3. Connect your WhatsApp account
4. Upload and set your new profile picture
